# CSE340 Assignment 3 - Vehicle Detail View

## Description
This project implements a vehicle inventory detail view using Node.js, Express, MVC, and PostgreSQL.

## Features
- Dynamic vehicle detail page based on `inv_id`
- Responsive design for all screen sizes
- 404 and 500 error handling
- Test link in footer for 500 error

## Setup
1. Install dependencies: `npm install`
2. Set up PostgreSQL database and update `database/connection.js`
3. Run the server: `npm start`
4. Visit: `http://localhost:3000/inventory/detail/1`
